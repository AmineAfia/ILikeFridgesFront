## Get Started

1 clone Project

```
git clone 
```

2 install dependencies

```
npm i
```

3 link dependencies for react-native (install it if needed)

```
rnpm link
```

4 run project

```
npm start
```
We used React Native Elements as a UI Library. The API is located [here](https://github.com/dabit3/React-Native-Elements).
